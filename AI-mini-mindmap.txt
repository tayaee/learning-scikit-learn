AI
    데이터(Data)
        빅데이터
            데이터 파이프라인 (Data Pipeline): ETL, ELT
            분산 처리 (Distributed Computing): Apache Hadoop, Spark
            대표 앱 및 사례: Google BigQuery, Snowflake
        데이터 전처리 (Data Preprocessing)
            결측치 처리 (Missing Value Imputation): 평균/중앙값 대체, MICE, KNN
            이상치 탐지 (Outlier Detection): IQR, Isolation Forest
            정규화/표준화 (Normalization/Standardization): Min-Max Scaling, Z-Score
            대표 앱 및 사례: Pandas (Python), OpenRefine
        데이터 증강 (Data Augmentation)
            이미지 (Image): 회전, 크기 조절, 색상 반전
            텍스트 (Text): 동의어 대체, 백번역(Back-translation)
            대표 앱 및 사례: Albumentations (이미지)
    머신러닝(ML)
        지도 학습(Supervised Learning)
            회귀(Regression)
            분류(Classification)
            대표 앱 및 사례: 스팸 필터링, 금융 사기 탐지, 의료 영상 분석(암 진단 등)
        비지도 학습(Unsupervised Learning)
            클러스터링(Clustering)
            차원 축소(Dimensionality Reduction)
            대표 앱 및 사례: 구글 뉴스 기사 분류, 고객 세분화(넷플릭스, 아마존 추천 시스템)
        강화 학습(Reinforcement Learning)
            핵심 개념: 에이전트, 환경, 상태, 행동, 보상
            대표 앱 및 사례: 자율 주행 자동차, 알파고(AlphaGo), 로봇 공학
        주요 프레임워크
            scikit-learn
    딥러닝(Deep Learning)
        신경망 구조(Neural Network Architectures)
            CNN (Convolutional Neural Network)
            RNN (Recurrent Neural Network)
            Transformer
        생성형 AI (Generative AI)
            GAN (Generative Adversarial Network)
            VAE (Variational Autoencoder)
            Diffusion Model
        NLP (Natural Language Processing)
            BERT
            GPT
            대표 앱 및 사례: ChatGPT, Google Gemini, Microsoft Copilot, Grammarly
        Agentic AI
            핵심 원리: Reasoning, Planning, Tool Use
            프레임워크: LangChain, LangGraph
            대표 앱 및 사례: Auto-GPT, BabyAGI
        모델 컨텍스트 프로토콜(MCP)
            목적: LLM과 외부 시스템 간의 상호작용 표준화
            기술적 내용: API 스펙, JSON 스키마
        주요 프레임워크 및 라이브러리
            딥러닝 프레임워크: TensorFlow, PyTorch, JAX
            LLM 개발: Hugging Face Transformers
            API: OpenAI REST API, Anthropic API
    컴퓨터 비전(Computer Vision)
        주요 기술: 객체 탐지(YOLO, R-CNN), 이미지 분할(Segmentation)
        대표 앱 및 사례: DALL-E, Midjourney, Stable Diffusion, Google Lens
    음성 AI(Speech AI)
        주요 기술: 음성 인식(ASR), 음성 합성(TTS)
        대표 앱 및 사례: Siri, Google Assistant, Amazon Alexa, Whisper(OpenAI)
    AI 보안 (AI Security)
        AI를 활용한 보안 기술
            악성코드 탐지: ML 모델 이용
            침입 탐지 시스템(IDS): 네트워크 트래픽 패턴 분석
            이상 행위 탐지(Anomaly Detection): 시스템 로그 분석
            스팸/피싱 탐지: 이메일 내용 분석
        AI 모델에 대한 공격 (Adversarial Attacks)
            적대적 예제 공격(Adversarial Examples)
            데이터 포이즈닝(Data Poisoning)
            모델 추출(Model Extraction)
        AI 모델 방어 기술 (Defensive Measures)
            적대적 학습(Adversarial Training)
            입력 검증(Input Sanitization)
            모델 워터마킹(Model Watermarking)
        애플리케이션 보안 테스트 (AST) 솔루션
            SAST (Static Application Security Testing)
                설명: 소스 코드 분석
                대표 제품: Checkmarx One, Veracode SAST
            DAST (Dynamic Application Security Testing)
                설명: 실행 중인 앱 공격 시뮬레이션
                대표 제품: Veracode DAST, Invicti, Burp Suite Professional
            RASP (Runtime Application Self-Protection)
                설명: 런타임에 실시간 공격 탐지/차단
                대표 제품: Contrast Security RASP, Imperva RASP
            IAST (Interactive Application Security Testing)
                설명: SAST+DAST 결합, 런타임 코드 분석
                대표 제품: Contrast Security IAST, Synopsys IAST
            SCA (Software Composition Analysis)
                설명: 오픈소스 취약점 분석
                대표 제품: Black Duck (Synopsys), Mend.io, Sonatype Nexus
            MAST (Mobile Application Security Testing)
                설명: 모바일 앱 특화 보안 테스트
                대표 제품: NowSecure, Appknox
            API Security Testing
                설명: API 취약점 점검
                대표 제품: Akamai API Security, Salt Security
